**이자제한법 기준 (2021.7.7 이후):**
- 최고이율 연 20% (이자제한법 제2조, 이자제한법 제2조제1항의 최고이자율에 관한 규정)

**계산 로직 — calculateLoanInterest(principal, agreedRatePercent, days):**
- effectiveRate = Math.min(agreedRatePercent, 20)
- interest = Math.floor(principal * (effectiveRate / 100) * (days / 365))
- excessInterest = agreedRatePercent > 20 ? Math.floor(principal * ((agreedRatePercent - 20) / 100) * (days / 365)) : 0
- 반환: { interest, effectiveRate, excessInterest, isExceeded: agreedRatePercent > 20 }

**입력 필드 (premium-card p-6):**
1. 대여 원금 (숫자, 천 단위 포맷)
2. 약정 이율(%) — type="number" min="0" max="200" step="0.1"
3. 대여 기간(일) — type="number" min="1"

**결과 표시 (premium-card p-6):**
- 적용 이율: effectiveRate% (초과 시 "약정 XX% → 제한 20%" 안내)
- 이자액: interest원 (category.color)
- 초과이율이 있을 경우: "이자제한법 위반 초과이자: XXX원 (무효)" — 빨간색(#ef4444) 텍스트로 경고

**계산 로직 — calculateUnjustEnrichment(principal, acquiredDate, returnDate):**
- 민법 제741조~제748조: 부당이득 반환 시 법정이율 연 5% 적용
- 지연일수 = Math.floor((returnDate - acquiredDate) / (1000*60*60*24))
- interest = Math.floor(principal * 0.05 * (days / 365))
- total = principal + interest
- 반환: { principal, interest, total, days }

**입력 필드 (premium-card p-6):**
1. 부당이득 원금 (숫자, 천 단위 포맷)
