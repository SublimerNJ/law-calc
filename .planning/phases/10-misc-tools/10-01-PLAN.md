**소멸시효 계산기** (`statute-of-limitations`) — MISC-01

`'use client'`. tool id = `statute-of-limitations`, category id = `misc`.

입력:
- 청구권 유형 select: 일반채권(10년), 상사채권(5년), 불법행위손해배상(3년/피해자인지일 기산), 임금채권(3년), 단기채권(1년 — 음식숙박/의사/약사/학교), 어음수표(3년/1년)
- 기산일 (date input, 권리 발생일 또는 피해 인지일)

계산 로직:
```
만료일 = 기산일 + 소멸시효 기간
남은 일수 = 만료일 - 오늘
```

결과:
- 소멸시효 기간
- 만료일 (YYYY년 MM월 DD일)
- 남은 기간 (N년 N개월 N일 또는 "이미 소멸")
- 법적 근거: 민법 제162조~제164조, 상법 제64조

날짜 계산은 JavaScript Date 사용. formatDate 함수로 `ko-KR` locale 표시.

**법률구조 대상 확인기** (`legal-aid`) — MISC-03

`'use client'`. tool id = `legal-aid`, category id = `misc`.

입력:
- 가구원 수 (1~6인 이상 select)
- 월 소득 (원)
- 재산 총액 (원)
- 지원 유형 select: 소송대리(무료법률구조), 법률상담, 형사변호

판정 로직 (대한법률구조공단 2026년 기준):
- 소송대리: 가구원 수별 기준 중위소득 125% 이하 (1인: 2,784,000원, 2인: 4,612,000원, 3인: 5,914,000원, 4인: 7,208,000원, 5인: 8,447,000원, 6인 이상: 9,681,000원) AND 재산 5억 이하
- 법률상담: 기준 중위소득 150% 이하 (위 금액 × 1.2)
- 형사변호: 기준 중위소득 125% 이하

결과:
- 지원 가능 여부 (색상 강조)
- 기준 초과 금액 (초과 시)

`'use client'`. tool id = `certified-letter`, category id = `misc`.

입력 (모두 textarea/input):
- 발신인 (이름, 주소)
- 수신인 (이름, 주소)
- 제목 (예: 채무 이행 촉구 내용증명)

- 발신일 (date, 기본값 오늘)

"미리보기" 버튼 클릭 시 결과 카드에 내용증명 양식 렌더링:

```
[내용증명]

발신인: {발신인이름}
       {발신인주소}

수신인: {수신인이름}
       {수신인주소}

제 목: {제목}

{본문}

위와 같이 내용을 증명합니다.

{발신일}

발신인: {발신인이름} (인)
```
