Statutory share fractions (민법 제1009조):
  Spouse + lineal descendants: spouse gets 1.5 share, each child gets 1 share

  Spouse + parents (no descendants): spouse gets 1.5, each parent gets 1
  No spouse: equal shares among same-rank heirs

Forced heirship fraction (유류분, 민법 제1112조):
  Lineal descendants: 법정상속분 × 1/2
  Spouse: 법정상속분 × 1/2
  Lineal ascendants: 법정상속분 × 1/3

```
function calculateInheritanceTax(grossEstate, hasSpouse, numChildren, funeralExpenses, debts):
  // 1. 과세가액 = 총상속재산 - 채무 - 장례비
  taxableBase = grossEstate - debts - Math.min(funeralExpenses, 15_000_000)

  // 2. 공제
  basicDeduction = 200_000_000
  spouseDeduction = hasSpouse ? Math.max(500_000_000, Math.min(grossEstate * 0.5, 3_000_000_000)) : 0
  lumpSumDeduction = 500_000_000  // 일괄공제

  // Use lump sum if > basic deduction
  totalDeduction = Math.max(lumpSumDeduction, basicDeduction) + spouseDeduction
  taxableAmount = Math.max(0, taxableBase - totalDeduction)

  // 3. Progressive tax
  tax = progressiveTax(taxableAmount)  // use 2026 rate table above
  return { taxableBase, totalDeduction, taxableAmount, tax }
```
