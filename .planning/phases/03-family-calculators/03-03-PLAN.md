Calculation logic (상속세 및 증여세법, 2026):
```typescript
function calculateGiftTax(giftAmount: number, priorGifts: number, relationship: string, isMinor: boolean): Result {
  // 1. Determine 10-year exclusion
  const exclusions: Record<string, number> = {
    '배우자': 600_000_000,
    '직계존속': isMinor ? 20_000_000 : 50_000_000,  // parent giving to child
    '직계비속': 50_000_000,  // child giving to parent
    '기타친족': 10_000_000,
  };
  const exclusion = exclusions[relationship] ?? 10_000_000;

  // 2. Taxable amount = (current gift + prior gifts within 10 years) - exclusion
  const combined = giftAmount + priorGifts;
  const taxableAmount = Math.max(0, combined - exclusion);

  // 3. Progressive tax on taxable amount
  const tax = progressiveTax(taxableAmount);

  // 4. Credit for tax already paid on prior gifts
  const priorTax = progressiveTax(Math.max(0, priorGifts - exclusion));
  const finalTax = Math.max(0, tax - priorTax);

  return { exclusion, combined, taxableAmount, tax: finalTax };
}
```
