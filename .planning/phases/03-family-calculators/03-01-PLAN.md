Calculation logic (민법 제839조의2):
- 분할대상 재산 = 혼인 중 형성된 공동재산 (혼인 전 재산, 상속/증여재산 제외)
- 기여도는 법원이 결정하나 기본 50:50, 전업주부 인정
- 청구인 취득액 = 분할대상재산 × 청구인기여도%
- 상대방 현 보유액이 청구인 취득액보다 많으면 → 차액을 상대방이 청구인에게 지급

Inputs:
1. 혼인 중 형성된 총 재산 (원) — text input with comma formatting
2. 청구인 명의 재산 (원) — text input
3. 상대방 명의 재산 (원) — text input (auto-fill = 총재산 - 청구인명의 if empty, but allow override)
4. 청구인 기여도 (%) — number input, default 50, range 20–80
5. 혼인기간 (년) — number input (informational, affects court's actual decision)

Result shows:
- 청구인 취득 예상액 (총재산 × 기여도%)
