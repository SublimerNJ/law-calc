  <files>scripts/migrate-theme.js</files>
  <action>
    - Create a Node.js script `scripts/migrate-theme.js` that recursively searches `src/app/tools/**/*.tsx`.
    - The script should replace common hardcoded dark theme tailwind classes with light theme equivalents:
      - `bg-[#0d1424]` -> `bg-white`
      - `border-[#1e2d4a]` -> `border-slate-200`
      - `text-white` -> `text-slate-900`
      - `text-gray-400` -> `text-slate-600`
      - `text-gray-300` -> `text-slate-600`
      - `text-gray-200` -> `text-slate-700`
      - `bg-[#1e2d4a]` -> `bg-slate-100` (often used for unselected tabs)
      - `focus:border-[#f59e0b]` -> `focus:border-blue-600` (changing to professional blue accent)
      - `focus:border-[#3b82f6]` -> `focus:border-blue-600`
      - `bg-[#f59e0b]` -> `bg-blue-600`
      - `text-black` -> `text-white` (when inside a blue selected tab/button)
      - `bg-gray-800` -> `bg-slate-100`
      - `bg-gray-900` -> `bg-white`
    - Run the script: `node scripts/migrate-theme.js`
  </action>
  <verify>
