**Korean law basis:** 대한변호사협회 변호사보수기준(2018 개정) — 소송비용에 산입되는 변호사 보수는 별도 기준으로 계산.

**실제 적용 기준 (민사소송규칙 제116조, 대법원 예규):**
소가(訴價) 구간별 산입 가능 변호사 보수:
- 2천만원 이하: 소가의 10% (최소 100,000원)
- 2천만원 초과 ~ 5천만원 이하: 200만원 + 2천만원 초과액의 8%
- 5천만원 초과 ~ 1억원 이하: 440만원 + 5천만원 초과액의 6%
- 1억원 초과 ~ 1.5억원 이하: 740만원 + 1억원 초과액의 4%
- 1.5억원 초과 ~ 2억원 이하: 940만원 + 1.5억원 초과액의 2%
- 2억원 초과 ~ 5억원 이하: 1,040만원 + 2억원 초과액의 1%
- 5억원 초과 ~ 10억원 이하: 1,340만원 + 5억원 초과액의 0.5%
- 10억원 초과: 1,590만원 + 10억원 초과액의 0.1% (최대 3,000만원 한도)

**입력 필드:**
1. 소가 (원, number input, placeholder "예: 50000000")
2. 심급 선택 (1심/2심/3심 radio — 2심은 1.5배, 3심은 1.2배 추가 산정 없음, 단 산입 비율 동일 적용)

**결과 표시:**
- 산입 가능 변호사 보수: XXX원 (천원 단위 콤마 포맷)
- 소가: XXX원
- 법적 근거 텍스트 (small)
