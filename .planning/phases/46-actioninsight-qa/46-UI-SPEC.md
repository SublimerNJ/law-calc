## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing (`space-y-3` roughly 12px, `mb-4` is 16px) |
| md | 16px | Default element spacing (`p-4`) |
| lg | 24px | Section padding (`md:p-6`, `mb-6`) |
| xl | 32px | Layout gaps (`mt-8`) |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: `p-5` (20px) used in `ActionInsight` component.
