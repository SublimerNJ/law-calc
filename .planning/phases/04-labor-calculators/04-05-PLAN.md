Formula (근로기준법 제2조제1항제6호):
- 1일 평균임금 = 산정사유 발생일 이전 3개월 임금총액 / 3개월 총 역일수
- 정기상여금: 산정 기간 중 지급된 상여금 × (3/상여지급주기(개월))로 포함

Inputs:
1. 산정 기준일 (date, default today) — 퇴직일, 재해일 등
2. 월별 임금 입력 (3개월):
   - 각 달: 기간 시작일, 종료일, 기본급(원), 각종수당(원)
   - 3행 자동 생성 (직전 3개월)
3. 해당 기간 지급된 정기상여금 (number, 원, optional)
4. 상여금 지급주기 (select: 1개월/2개월/3개월/6개월/12개월, optional, default 6)

Derived:
- 3개월 총 역일수 = sum of days in each monthly period
- 3개월 임금총액 = sum(기본급 + 수당) + 상여금 × 3 / 지급주기
- 1일 평균임금 = 총액 / 총역일수
- 보정: 최저 1일 평균임금 >= 통상임금을 밑돌지 않음 (표시만)

Results:
- 1일 평균임금 (large, category color)
- 3개월 임금총액
- 3개월 총 역일수
- 산정 기간 (YYYY.MM.DD ~ YYYY.MM.DD)
- 상여금 포함 여부

Legal basis: 근로기준법 제2조제1항제6호(평균임금 정의), 제6조(균등처우)

Tool: `TOOLS.find(t => t.id === 'shutdown-allowance')`, category labor.

Formula (근로기준법 제46조):
- 기본: 1일 평균임금 × 70% × 휴업일수
- 단서: 위 금액이 통상임금보다 적으면 통상임금을 기준으로 지급 가능
- 노동위원회 승인 시: 평균임금 70% 미만 지급 가능 (특수사정)

Inputs:
1. 1일 평균임금 (number, 원) — 직접 입력 (평균임금 계산기 사용 후 옮겨 입력)
2. 월 통상임금 (number, 원) — 비교 기준
3. 휴업일수 (number, 일)
4. 노동위원회 승인 여부 (checkbox) — 승인 시 별도 비율 입력 가능
5. 승인 지급률 (number, %, 노동위 승인 시 활성화)

Derived:
- 1일 통상임금 = 월통상임금 / 30
- 평균임금 70% 기준 = 1일평균임금 × 0.7
- 비교: 평균임금70% vs 통상임금
- 적용기준 = 노동위승인없을때 max(평균임금70%, 통상임금)
  NOTE: 실무상 사용자는 평균임금 70% OR 통상임금 중 선택(근로자에 불리하지 않은 범위); 계산기는 법적 최저(평균임금70%)와 통상임금 둘 다 표시
- 총 휴업수당 = 적용기준 × 휴업일수

Results:
- 총 휴업수당 (large)
- 평균임금 70% 기준 총액
- 통상임금 기준 총액 (비교)
- 적용 기준 안내 (어느 쪽이 높은지)
- 1일 지급액

Note: "사용자 귀책사유 없는 휴업(천재지변 등)은 노동위원회 승인 후 지급률 조정 가능"

Legal basis: 근로기준법 제46조(휴업수당)
